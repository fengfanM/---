# 天机抽卡 - 游戏策划文档

> 算命 · 占卜 · 抽卡 · 本地存档（纯前端）

---

## 1. 游戏概述

### 1.1 核心玩法
天机抽卡是一款融合了东方玄学、抽卡收集和日常占卜的休闲游戏。玩家通过灵石进行抽卡，收集不同稀有度的卡牌，完成每日/周任务，解锁成就，并体验每日运势占卜。

### 1.2 游戏特色
- **纯前端本地存档**：所有数据存储在浏览器本地，无需服务器
- **东方美学风格**：采用古朴典雅的视觉设计
- **多系统融合**：抽卡、收集、成就、任务、占卜一体
- **每日循环体验**：每日运势、每日任务、连续签到

## 2. 核心系统设计

### 2.1 抽卡系统

#### 2.1.1 基础规则
- 单抽消耗：10 灵石
- 十连消耗：90 灵石（节省 10 灵石）
- 每日免费抽卡：1 次
- 保底机制：
  - SR 保底：10 抽内必出 SR 及以上
  - SSR 保底：60 抽内必出 SSR

#### 2.1.2 概率分布
| 稀有度 | 基础概率 | 权重 |
|--------|----------|------|
| SSR    | 2%       | 20   |
| SR     | 15%      | 150  |
| R      | 35%      | 350  |
| N      | 48%      | 480  |

#### 2.1.3 卡池系统
支持多个卡池切换，每个卡池有独立的权重调整。

#### 2.1.4 保底进度提示
- SR 剩余≤3 次时：进度条高亮、文字变色（#ff6b6b）、脉冲动画
- SSR 剩余≤15 次时：进度条高亮、文字变色、脉冲动画
- 显示"即将保底！"提示

### 2.2 卡牌系统

#### 2.2.1 卡牌稀有度
| 稀有度 | 颜色 | 边框 | 描述 |
|--------|------|------|------|
| N      | 灰色 | 普通 | 普通卡牌 |
| R      | 蓝色 | 稀有 | 稀有卡牌 |
| SR     | 金色 | 史诗 | 史诗卡牌 |
| SSR    | 紫金 | 传说 | 传说卡牌 |

#### 2.2.2 卡牌属性
- 元素属性：星、月、日、空、风、水、火、山
- 关键词：卡牌特色标签
- 简介：简短描述
- 背景故事：详细设定

#### 2.2.3 卡牌图鉴
- 收集进度百分比
- 收集里程碑奖励：
  - 25%：500 灵石
  - 50%：1000 灵石
  - 75%：2000 灵石
  - 100%：5000 灵石

### 2.3 灵石系统

#### 2.3.1 灵石获取途径
| 途径 | 数量/频率 |
|------|-----------|
| 每日签到 | 20-200（递增） |
| 每日任务 | 50-300 |
| 周任务 | 100-1000 |
| 成就奖励 | 50-1000 |
| 图鉴奖励 | 500-5000 |

#### 2.3.2 每日签到规则
- 每日可签到 1 次
- 奖励递增机制：
  - 第 1 天：20 灵石
  - 第 2 天：30 灵石
  - 第 3 天：40 灵石
  - 第 4 天：60 灵石
  - 第 5 天：80 灵石
  - 第 6 天：120 灵石
  - 第 7 天：200 灵石
- 连续签到满 7 天后重置为 20 灵石开始新一轮

### 2.6 占卜系统

#### 2.6.1 今日运势
- 每日可生成一次运势
- 包含：签运、气运、幸运元素、幸运数、指引
- 支持轮盘buff影响

#### 2.6.2 三牌阵（过去/现在/未来）
- 抽取三张卡牌进行占卜
- 自动生成占卜叙事
- 结合卡牌关键词生成解读

#### 2.6.3 命运轮盘
- 每日可转动一次
- 提供各种buff效果
- buff持续到当天结束

### 2.7 统计系统

#### 2.7.1 核心统计数据
- 总抽卡次数
- SSR 获得数
- SR 获得数
- 抽卡概率分布
- 距上次 SSR 抽卡次数
- 距上次 SR 抽卡次数
- 今日运势查看次数
- 命运轮盘转动次数
- 三牌占卜次数
- 任务完成数
- 是否触发过 SSR 保底

#### 2.7.2 抽卡历史
- 记录最近 50 次抽卡
- 包含时间戳、卡牌名称、稀有度
- 支持查看历史概率分布

## 3. 音效系统

|--------|----------|
| tap | 按钮点击、抽卡按钮 |

### 3.2 音效实现
使用 Web Audio API 合成音效，无需外部音频文件。

## 6. 数值平衡

### 6.1 灵石消耗与产出
- 每日产出：约 200-400 灵石
- 每日消耗：约 100-200 灵石
- 周产出：约 1500-3000 灵石
- 周消耗：约 700-1400 灵石

### 6.2 抽卡频率
- 轻度玩家：每日 1-3 抽
- 中度玩家：每日 5-10 抽
- 重度玩家：每日 20+ 抽

### 6.3 SSR 获得预期
- 平均 50 抽获得一张 SSR
- 保底 60 抽必出 SSR
- 预期每周获得 1-2 张 SSR

## 8. 技术实现

### 8.1 技术栈
- React 18 + TypeScript
- Zustand（状态管理）
- Vite（构建工具）
- PIXI.js（抽卡动画）

### 8.2 核心文件
- `src/App.tsx` - 主应用组件
- `src/store/gameStore.ts` - 游戏状态管理
- `src/types.ts` - 类型定义
- `src/data/cards.json` - 卡牌数据
- `src/features/gacha/gachaEngine.ts` - 抽卡引擎
- `src/lib/gameAssets.ts` - 音效和资源
